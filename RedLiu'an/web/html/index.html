地方概况
大别山区域中心城市
现辖金安、裕安两区和霍邱、金寨、霍山、舒城四县，以及省级六安经济技术开发区和叶集改革发展试验区。
上古时，这里是偃姓皋陶部族活动和聚居地。春秋、战国时期属楚。
隋时，分属淮南郡和庐江郡。唐时，分属寿州和庐州，其间置舒城县和盛唐县。
明时，北属寿州、南属六安州。清初，属江南行省右布政使司所辖的寿州和六安州。
中华民国初年，寿县、霍邱属安徽省淮泗道，其余县区属安徽省安庆道。
1949年1月，六安解放，4月，成立六安地区专员公署，隶属皖北行署。
